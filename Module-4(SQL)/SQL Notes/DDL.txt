DDL => Data Definition Language

Commands :- CREATE, ALTER, DROP

1. CREATE :-

1. Create a Database :- CREATE DATABASE dc09db;

2. Create Table :-
=>
CREATE TABLE Student(StudentId INT PRIMARY KEY AUTO_INCREMENT, email varchar(50), address varchar(100), zipcode varchar(10));
CREATE TABLE Courses(cid INT PRIMARY KEY AUTO_INCREMENT, cname varchar(50), fees float, duration varchar(20));

2. ALTER Table :-

=> column ne add karva mate :-
ALTER TABLE Student ADD COLUMN contact varchar(50);

=> Exisiting column ne modify or change karva mate :-
ALTER TABLE Student CHANGE contact Phone int;

=> Exisiting column ne Drop karva mate
ALTER TABLE Student DROP COLUMN Phone;

=> Whole table ne DROP karva mate
DROP TABLE Student;
